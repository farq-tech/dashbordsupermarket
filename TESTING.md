# 🧪 دليل التجربة والاختبار

## 🚀 تشغيل التطبيق

```bash
# 1. تثبيت المكتبات (إذا لم يتم بعد)
npm install

# 2. تشغيل التطبيق
npm run dev
```

التطبيق سيفتح تلقائياً على `http://localhost:3000`

## 📋 خطوات الاختبار

### 1. تحديد الموقع
- عند فتح التطبيق، سيطلب منك السماح بالوصول للموقع
- اضغط على "السماح" (Allow)
- إذا فشل تحديد الموقع، يمكنك الضغط على "إعادة المحاولة"

### 2. تصفح التصنيفات
- في الأعلى، ستجد شريط التصنيفات
- اضغط على أي تصنيف لتصفية المنتجات
- اضغط على "الكل" لعرض جميع المنتجات

### 3. إضافة منتجات للسلة
- كل منتج يحتوي على:
  - الصورة (من `devapi.fustog.app/img/{id}.jpg`)
  - الاسم بالعربي
  - السعر (من API ItemsPrices)
  - أزرار التحكم بالكمية (+/-)
  - زر "إضافة"
- اختر الكمية المطلوبة
- اضغط على "إضافة" لإضافة المنتج للسلة
- ستظهر رسالة "✓ تمت الإضافة" عند النجاح

### 4. فتح السلة
- اضغط على زر "🛒 السلة" في الأعلى
- ستفتح سلة المشتريات من الجانب الأيمن
- في السلة يمكنك:
  - تعديل الكمية لأي منتج
  - حذف منتج (🗑️)
  - إفراغ السلة بالكامل
  - مشاهدة المجموع الكلي

### 5. إرسال الطلب
- اضغط على "إرسال الطلب" في السلة
- ستفتح نافذة تأكيد الطلب
- راجع:
  - ملخص الطلب والمنتجات
  - موقع التوصيل
  - المجموع الكلي
- يمكنك إضافة ملاحظات (اختياري)
- اضغط على "تأكيد وإرسال"
- ستظهر رسالة نجاح مع:
  - رقم الطلب
  - وقت التوصيل المتوقع

## 🔧 جلب البيانات من API

التطبيق يستخدم الـ APIs التالية:

### 1. Categories API
```
GET https://api.fustog.app/api/v1/category/Categories
```
- يجلب جميع التصنيفات

### 2. Products API
```
GET https://api.fustog.app/api/v1/product/ProductsByCategory?categoryId={id}
```
- يجلب المنتجات حسب التصنيف

### 3. **ItemsPrices API** (الجديد!)
```
GET https://api.fustog.app/api/v1/product/ItemsPrices
```
- يجلب أسعار جميع المنتجات من جميع المتاجر
- التطبيق يربط الأسعار تلقائياً مع المنتجات
- يختار أقل سعر إذا كان المنتج متوفر في عدة متاجر

### 4. Images
```
https://devapi.fustog.app/img/{imageId}.jpg
```
- يعرض صور المنتجات

## 📱 اختبار Responsive

جرب التطبيق على:
- **Desktop**: عرض كامل مع grid واسع
- **Tablet**: عرض متوسط
- **Mobile**: عرض مُحسّن للموبايل
  - السلة تأخذ كامل العرض
  - الشريط scrollable
  - Products grid يتكيف مع حجم الشاشة

## 🐛 استكشاف الأخطاء

### المنتجات لا تظهر
1. تحقق من تحديد الموقع
2. افتح Console (F12) لرؤية أي أخطاء API
3. التطبيق سيستخدم Mock Data تلقائياً إذا فشل API

### الصور لا تظهر
- الصور تأتي من `devapi.fustog.app`
- تحقق من الاتصال بالإنترنت
- تحقق من Console لأي أخطاء CORS

### الأسعار غير صحيحة
- الأسعار تأتي من `ItemsPrices` API
- إذا لم يكن للمنتج سعر في API، سيستخدم السعر الافتراضي من بيانات المنتج
- في Mock Data، جميع الأسعار محددة مسبقاً

## 📊 مراقبة Network Requests

افتح Developer Tools (F12) → تبويب Network:
- `Categories`: جلب التصنيفات
- `ProductsByCategory`: جلب المنتجات لكل تصنيف
- `ItemsPrices`: **جلب جميع الأسعار**
- `img/*.jpg`: تحميل الصور
