## 🔧 الكود الحالي

### معالجة Response الفارغ:
```python
if method == "POST":
    resp = self._client.post(url, params=params, timeout=30.0)
    # إذا كان الـ response فارغ، ارجع قائمة فارغة
    if resp.status_code == 200 and len(resp.content) == 0:
        print(f"[WARN] Empty response from {url} - API returned 200 but no content")
        return []  # ارجع قائمة فارغة بدلاً من رفع خطأ
```
